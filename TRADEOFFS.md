# Tradeoffs

## Authentication

Authentication was skipped to focus on:
- ingestion workflow
- review system
- deployment

---

## Simplified Normalization

Normalization logic is simplified for demo purposes.

Real ESG systems would include:
- advanced mapping
- conversion pipelines
- validation layers

---

## CSV-Only Input

The platform currently supports CSV uploads only.

Future versions may support:
- SAP APIs
- Excel files
- ERP integrations
