# Technical Decisions

## Why Django REST Framework

Used DRF for:
- rapid API development
- serializer support
- API testing
- scalable backend architecture

---

## Why PostgreSQL

Used PostgreSQL because:
- strong relational support
- JSON field support
- production-ready database

---

## Why React + Vite

Used React and Vite because:
- fast frontend development
- simple component structure
- lightweight deployment

---

## Why Render + Vercel

Render:
- simple Django deployment

Vercel:
- easy React deployment
- automatic CI/CD
